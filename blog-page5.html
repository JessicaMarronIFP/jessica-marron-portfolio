Aplicación desarrollada para un cliente con Softr, una plataforma No Code que permite crear aplicaciones web dinámicas sin necesidad de programación tradicional. El proyecto incluye la estructuración de bases de datos, configuración de roles y permisos, y creación de vistas personalizadas para distintos tipos de usuario.

Además, se implementaron automatizaciones y workflows desde Softr y Make para la gestión eficiente de tareas, actualización automática de registros y optimización de procesos internos, mejorando la productividad y reduciendo el trabajo manual.

Softr · Make · Bases de datos relacionales · Automatizaciones · Workflows · Modelado de datos · Gestión de roles · UX/UI

El proyecto consistió en el diseño e implementación de una aplicación empresarial con Softr, incluyendo la creación y estructuración de la base de datos, la configuración de roles y permisos y el desarrollo de automatizaciones con Make. Además, implicó trabajo directo con el cliente para la recogida de requisitos y adaptación de la solución a necesidades reales.
